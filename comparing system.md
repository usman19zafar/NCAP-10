
1. Factor Analysis (FA) — Why it helps NCAP‑10
FA is NOT useful for neuromorphic cognition directly.
But it teaches us the correct way to define constructs.

Human tests use FA to discover:

Fluid Reasoning

Knowledge

Quantitative Reasoning

Visual‑Spatial

Working Memory

NCAP‑10 uses the same idea, but with different constructs:

Stability

Synchrony

Energy flow

Attractor formation

Plasticity

Timing

Glial‑field modulation

Synaptic favorability

Mutable memory

Dynamic reasoning

FA teaches us:

how to define constructs cleanly

how to avoid overlap

how to ensure each construct measures something unique

how to validate that constructs are real

We borrow the method, not the factors.

⭐ 2. Item Response Theory (IRT) — Why it helps NCAP‑10
IRT is the most advanced scoring engine in human psychometrics.

It models:

difficulty

discrimination

guessing

information curves

NCAP‑10 replaces IRT with a dynamical equivalent:

difficulty → energy cost

discrimination → stability sensitivity

guessing → noise floor / stochastic drift

information → dynamical information gain

IRT teaches us:

how to model task difficulty

how to model system ability

how to design tasks that reveal differences between systems

how to build adaptive testing

We borrow the logic, not the equations.

⭐ 3. Norming + Standardization — Why it helps NCAP‑10
Human tests compare people to age groups.

NCAP‑10 compares architectures, configurations, and versions.

Norming teaches us:

how to make scores comparable

how to build benchmarks

how to define “normal performance”

how to detect outliers

how to track improvement over time

In NCAP‑10, norms are based on:

system class

hardware configuration

training regime

energy budget

memory architecture

glial‑field settings

We borrow the concept, not the tables.

⭐ The compressed truth
FA → teaches us how to define constructs
IRT → teaches us how to score performance
Norms → teach us how to compare systems

But NCAP‑10 replaces:

FA factors → dynamical constructs

IRT difficulty → energy/stability/synchrony cost

Norms → architecture‑based benchmarks

We use the psychometric architecture, not the psychometric content.

⭐ The 5‑year‑old version
Human tests sort puzzles, measure difficulty, and compare people.
NCAP‑10 sorts brain‑like behaviors, measures energy and stability, and compares systems.
